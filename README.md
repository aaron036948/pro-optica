## 📖 Descripción del Proyecto
Este proyecto consiste en el desarrollo de un sistema informativo, es un catalago de lentes que permite al usuario ver los precios de los lentes y sus modelos.

## ⚙️ Instalación y Ejecución

### Sistema de inventario carpetaprincipal proyecto2_javascript

1. Clonar el repositorio:
https://github.com/carolmilena12/proyecto2_javascript

2. Entrar a la carpeta del proyecto:
cd proyecto2_javascript

3. Entrar a la carpeta del backend:
cd backend

4. Instalar dependencias:
npm install

5. Ejecutar el servidor:
node server js

👉 El pagina se ejecutará en: http://localhost:3000
